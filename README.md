# DIY Tools

A collection of handy calculation and utility tools for your everyday projects. Built with React, TypeScript, Vite, and Tailwind CSS. Installable as a PWA.

## Tools

- **Pythagoras Calculator** — Calculate the hypotenuse or legs of a right triangle with real-time visualization.
- **Circle Chord Calculator** — Calculate the straight distance between adjacent points on a circle given the radius and number of points.
- **Rule of Three** — Solve proportions: given any three values of a proportion `A : B = C : X`, find the fourth.

## Usage

```bash
npm install
npm run dev       # Start dev server at http://localhost:5173
npm run build     # Production build to dist/
npm run preview   # Preview production build locally
```
